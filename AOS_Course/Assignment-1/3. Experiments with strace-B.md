## Verify the command
```bash
ls /usr/include | grep "\.c"
```
Should see all files ending in `.c` inside `/usr/include`

## Trace the entire pipeline

```bash
strace -f -o trace.txt sh -c 'ls /usr/include | grep "\.c"'
```

### Explanation

- `strace` → Trace system calls.
- `-f` → Follow child processes (`ls` and `grep`).
- `-o trace.txt` → Save the trace in a file.
- `sh -c` → Execute the entire pipeline through a shell.
- `'ls /usr/include | grep "\.c"'` → The command we're tracing.
