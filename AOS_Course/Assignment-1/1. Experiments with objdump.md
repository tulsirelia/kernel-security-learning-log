## Creating the file
nano hello.c

write the code

ctrl+O 

check the file using ls, it must be there

then compile is using 
```bash
gcc -c hello.c
```
-c only compiles, do not produce an executable

again check the files using ls, should see 'hello.c' and 'hello.o'

<img width="943" height="192" alt="image" src="https://github.com/user-attachments/assets/5e3ee91d-2bb1-42b2-92d3-2c5b4e1d603a" />

we can't open hello.o with cat, which will show us garbage because it contains machine code.

```
Disassembly means converting machine code into assembly language
```

**C Code ➡️ Compiler ➡️ Machine code (.o) ➡️ Disassembler ➡️ Assembly Code**

So we generate it using objdump
```bash
objdump -d hello.o
```
-d means disassemble
